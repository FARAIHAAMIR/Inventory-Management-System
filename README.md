# Inventory-Management-System

Project Overview

This project outlines the design and feature set for an Inventory Management System (IMS) that serves two types of businesses: offline retail shops and online e-commerce platforms. The system is built around two core user roles. Shop Owner (or Platform Manager) and Rider and is designed to scale from manual operations in Phase 1 toward automation and advanced analytics in future phases. The platform begins with a public-facing landing page and branches into role-specific dashboards after authentication, keeping the experience clean and focused for each type of user.
